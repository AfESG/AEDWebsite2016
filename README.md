# AED
Frontend scaffold: Webpack, Babel, Stylus (autoprefixer), React, Redux, ESLint

## Develop
* `npm install`
* `npm install -g webpack-dev-server`
* `npm start`

## Test
* `npm test`

## Build
* Set `['output']['publicPath']` to your deployment URL in `webpack-production.config.js`
* npm run build
